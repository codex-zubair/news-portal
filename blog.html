What is the difference between var, const and let in JavaScript?

var is the traditional way of declaring variables in JavaScript and it is globally scoped, it is considered a bad practice in JavaScript to declare variables with var nowadays. let and const were introduced as a new way to declare variables with ES6. With let, you may reassign the value but the variable will only be used inside the block it was defined in and not outside, kind of like having its own scope. you cannot reassign the value if you declared the variable as const(not to be confused with immutable, a const object can have properties mutated)

What is difference between arrow function and normal function?

Arrow functions are a new way to write anonymous function expressions. Unlike regular functions, arrow functions do not have their own this. There are two major benefits of using Arrow functions. One is that it's a shorter syntax and thus requires less code. The main benefit is that it removes the several pain points associated with the this operator.

differences between map and foreach?

The forEach() method does not create a new array based on the given array. The map() method creates an entirely new array. The forEach() method returns “undefined“. The map() method returns the newly created array according to the provided callback function.

Difference between filter and find in javascript?

The find() method is used to find all the descendant elements of the selected element. It finds the element in the DOM tree by traversing through the root to leaf. The filter() method is used to filters all the elements and returns the element that matches and the element that do not match are removed.

why we use template string in javascript?

Template literals are literals delimited with backtick (`) characters, allowing for multi-line strings, for string interpolation provide an easy way to interpolate variables and expressions into strings. and for special constructs called tagged templates.
